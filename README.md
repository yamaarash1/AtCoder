> C++ runner on Docker

# Usage

## image build

```
$ make
```

## project setup

```
$ make new path=ABC173/A
```

## run

```
$ make run
```

## test

```
$ make test
```

## switch project

```
$ make set path=ABC173/A
```

## clean

```
$ make clean
```

For more information, please see Makefile.
https://github.com/siu-issiki/cpp-docker/blob/master/Makefile
